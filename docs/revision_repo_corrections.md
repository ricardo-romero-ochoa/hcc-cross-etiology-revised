# Repository corrections for major revision

This update addresses the repository/reproducibility items raised during the major revision.

## Changes made

1. Added `REPOSITORY_MANIFEST.csv` to map reviewer-requested reproducibility objects to repository paths.
2. Added `scripts/16_revision_repository_audit.R` with source-only and strict audit modes.
3. Corrected `scripts/run_all.R` path errors:
   - `scripts/15_make_supplementary_tables.R`
   - removed the non-existing audit call from the automatic core pipeline; the final audit is run manually.
4. Added missing dependencies to `scripts/00_install_packages.R`:
   - `scales`, `officer`, `flextable`.
5. Corrected the `HBV_INJURY_TOP_2000` role label from `compact_primary_candidate` to `compact_primary`.
6. Made `scripts/12_make_manuscript_tables.R` backward-compatible with either `compact_primary` or `compact_primary_candidate`.
7. Added `scripts/06b_tcga_survival_cox_from_cached_inputs.R` to regenerate `results/tables/tcga_lihc_survival_cox_models.tsv` from cached TCGA module-score/clinical data without re-downloading TCGA.
8. Updated `.gitignore` so reviewer-facing generated tables and figures are not excluded from the release.
9. Updated `README.md` with the manifest, audit, and TCGA Cox-table recovery instructions.
10. Added `results/tables/inclusion_exclusion_table.tsv` as a compatibility alias generated by `scripts/01_dataset_inventory.R`.

## Final recommended sequence before release

```bash
Rscript scripts/00_install_packages.R
Rscript scripts/run_all.R

# If TCGA Cox table is missing but cached TCGA score/clinical table exists:
Rscript scripts/06b_tcga_survival_cox_from_cached_inputs.R

# After running CIBERSORTx externally and saving the result file:
Rscript scripts/09_cibersortx_adjusted_regression_GSE121248.R
Rscript scripts/11b_make_missing_figures.R
Rscript scripts/12_make_manuscript_tables.R
Rscript scripts/13_summarize_revision_results.R
Rscript scripts/14_make_supplementary_figures.R
Rscript scripts/15_make_supplementary_tables.R

# Final reviewer-facing audit:
Rscript scripts/16_revision_repository_audit.R --strict
```
